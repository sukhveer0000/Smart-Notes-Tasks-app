# NoteTask Manager

Smart Android app to create, manage, and organize both notes and tasks efficiently. Lightweight, easy-to-use, and designed for daily productivity.

## Features
- Add, edit, and delete notes
- Add, edit, and delete tasks
- Organize notes/tasks by date
- Swipe to delete with undo option
- Simple and intuitive interface

## Installation

### 1. Clone the repository
git clone (https://github.com/sukhveer0000/Smart-Notes-Tasks-app.git)


### 2. Open in Android Studio
- Android Studio → Open  
- Select the project folder  
- Wait for Gradle sync

### 3. Run the App
- Connect device OR run emulator  
- Press the Run ▶ button

### 4. (Optional) Build APK
Build > Build APK(s)
