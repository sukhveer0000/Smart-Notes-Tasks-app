# NoteTask Manager

Smart Android app to create, manage, and organize both notes and tasks efficiently. Lightweight, easy-to-use, and designed for daily productivity.

## Features
- Add, edit, and delete notes
- Add, edit, and delete tasks
- Organize notes/tasks by date
- Swipe to delete with undo option
- Simple and intuitive interface
